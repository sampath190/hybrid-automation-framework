# Hybrid Automation Framework

## Overview
A Selenium WebDriver automation framework developed using Java, TestNG, and Page Object Model (POM).

## Tech Stack
- Java
- Selenium WebDriver
- TestNG
- Maven
- Page Object Model (POM)
- Extent Reports
- Log4j2

## Features
- Cross-browser support
- Page Object Model design
- Data-driven testing
- Extent Reports
- Screenshot capture on failure
- Logging using Log4j2
- TestNG Groups and XML execution

## Test Scenarios Automated
- Login Validation
- Dashboard Validation
- Data Driven Login
- Logout Validation
- Admin Page Validation
- Add Employee
- Search Employee
- Delete Employee
- Edit Employee

## Project Structure
src/test/java
├── pageObjects
├── testCases
├── testBase
├── utilities

## How to Run
1. Clone the repository
2. Open in Eclipse
3. Update Maven dependencies
4. Execute testng.xml
